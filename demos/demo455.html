Scrawl multiFilters - pixelate

Extensions: images, filters, animation

The pixelate filter accepts four attributes: width, height, offsetX and offsetY. The offsets should not be larger than their respective dimension values, and all values should be positive integers.

This filter caches precalculated data in the multiFilter cache - MultiFilter.cache.pixelate - when updating the filter's attributes dynamically this cache needs to be set to false so the filter knows it needs recalculating.
