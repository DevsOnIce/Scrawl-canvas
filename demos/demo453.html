Scrawl multiFilters - filters which take color channels arguments

Extensions: images, filters, animation

The 'default' filter does not apply any filter engines to the parrot Picture entity. It does, however, make the entity use the multiFilter stamp code rather than the normal code. The Wheel entity, which is stamped on the cell before the parrot, has no multiFilter set on it.

The other filters in this demo all accept three additional attributes - red, green and blue

  • For the channels filter, these range from 0+ with 1 = normal, <1 = less of that color and >1 = more of that color
  • For the channelstep filter, values range from 1 (256 steps for that channel - normal) to 127 (only 2 steps for that channel)
