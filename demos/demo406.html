Scrawl multiFilters - matrix

Extensions: images, multifilters, animation

The matrix filter accepts four attributes: width, height, offsetX and offsetY. The offsets are relative to the current pixel during the calculations, so for a 5 x 5 matrix centred on the current pixel, the offset values should be -2. The filter also requires a weights attribute; the number of items in the weights array must match the volume of the matrix (width * height).
