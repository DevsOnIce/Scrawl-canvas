Scrawl.js - demonstration 003

Importing a canvas into the Scrawl library

Modules: phrase

It looks like your browser doesn't support the <canvas> element.

Scenario: automatically detect a canvas element defined in the DOM, and populate it with sprites

Actions:

  1. detect the existence of a canvas element via scrawl.init(), scrawl.pageInit() and scrawl.getCanvases()
  2. create several phrase sprites using factory and clone functions, and associate them with canvas element's Cell wrapper
  3. display phrase sprites on canvas

Known browser issues: none
