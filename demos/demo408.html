Scrawl multiFilters - chroma key (green screen effect)

Extensions: images, multifilters, animation

The chroma key filter produces a green screen effect, rendering pixels which fall between two color values as transparent. More than one range of colors can be defined in the filter. The ranges attribute takes an Array value, with each color range defined as an array within that array.

The color range array takes the form [startRed, startGreen, startBlue, endRed, endGreen, endBlue], with each value being an integer between 0 and 255. Start values for each color must be below end values - for speed, the filter does not check that values have been ordered correctly. In this demo, the start values of a single color range have been set to 0, while end values can be controlled by the range selectors.

startRed startGreen
startBlue endRed
